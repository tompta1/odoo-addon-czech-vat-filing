Czech VAT Filing
================

Generate Czech VAT filing exports for Odoo 19.

This addon provides XML export for:

* ``DPHDP3``
* ``DPHKH1``
* ``DPHSHV``

For the full project documentation and operator-focused setup notes, see:

* repository ``README.md``
* ``addons/custom/l10n_cz_vat_filing/README.md``
